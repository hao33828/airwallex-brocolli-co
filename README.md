# Brocolli & co.

 - This project is written in Next.js for fast loading

### Getting Started

For Development:
```
  npm run dev
```

For Production:
```
  npm run build
```

For Unit Tests:
```
  npm run test
```

For ESLint:
```
  npm run lint
```

For Typescript Check:
```
  npm run type-check
```
